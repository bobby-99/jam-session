# Jam Session

### Real-time collaborative playlist where guests vote to shape the music.

## Tech Stack

### Frontend

- React 18
- Vite 5
- React Router v6
- Tailwind CSS
- Axios
- Socket.io client

### Backend

- Node.js (ESM)
- Express
- Socket.io
- Zod validation
- Axios (OAuth + YouTube Data API)
- JSON Web Token

### Data and infrastructure

- PostgreSQL 16
- SQL migration runner (`schema_migrations` table)
- Docker Compose for local DB
